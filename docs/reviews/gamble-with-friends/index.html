Experience

Getting into Gamble With Friends wasn’t the problem.

Staying connected was.

The game launched successfully and reached the main menu without any issues, but every attempt to connect to the Steam servers failed. Since the core experience revolves around playing online, I wasn’t able to meaningfully test the game beyond its menus.

Technically, it boots. Practically, there’s nothing to play.

smol-eggo’s Verdict

You can’t really review a multiplayer game when your biggest opponent is the connection screen.

Hopefully this is something future updates can solve.

Did I forget I was testing?

No.

The server errors made sure of that.
