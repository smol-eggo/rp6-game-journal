Experience

From a technical standpoint, World of Horror performs exactly as you’d hope.

Gameplay was smooth throughout my testing, with no noticeable performance issues. Unfortunately, the same can’t be said for the controls. Navigating menus and interacting with the interface never felt particularly comfortable. It’s a game clearly designed with a mouse in mind, and that shows.

None of this makes the game unplayable, but it does make it harder to fully recommend over playing it on a PC.
