Experience

I launched Tetris Effect to see how it would perform on the RP6.

Three hours later I remembered I was supposed to be testing compatibility.

The game runs beautifully and feels perfectly suited for a handheld. Performance stayed smooth throughout my session, and I never felt the need to tweak settings or think about the hardware.

I simply got lost in the music, visuals, and “one more stage.”
